# CogniSense AI - Sistem de Suport Decizional pentru Psihoterapie

## Descriere
Această aplicație utilizează modele Transformer (RoBERTa) pentru a detecta distorsiunile cognitive din discursul pacienților și a genera profiluri clinice automate.

## Structura Proiectului
- `src/`: Codul sursă al aplicației (Streamlit).
- `models/`: Modelele AI antrenate (salvate local).
- `data/`: Fișiere de feedback și log-uri.

## Instalare și Rulare
1. Instalați dependențele:
   `pip install -r requirements.txt`
2. Porniți aplicația:
   `cd src`
   `streamlit run app.py`
